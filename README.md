# ELITE CHAT 💬

A real-time messaging application with one-on-one and group chats, file/image sharing, voice/video calls, and more.

## Features ✨

- **One-on-One Messaging**: Direct private conversations
- **Group Chats**: Create and manage group conversations
- **File & Image Sharing**: Share media files seamlessly
- **Voice & Video Calls**: Real-time audio and video calling
- **User Authentication**: Secure login and registration
- **Message History**: Persistent message storage
- **Typing Indicators**: See when others are typing
- **Read Receipts**: Know when messages are read
- **Real-time Updates**: WebSocket-powered instant notifications
- **Cross-platform**: Web and mobile applications

## Tech Stack 🛠️

- **Frontend (Web)**: React, Redux, Tailwind CSS
- **Frontend (Mobile)**: React Native, Redux, NativeWind
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Real-time Communication**: WebSocket (Socket.io)
- **Authentication**: JWT
- **Media Handling**: Multer, Sharp
- **Video/Audio**: WebRTC, Jitsi
